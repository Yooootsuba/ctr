# ctr

ctr is a number base converter in terminal

## Build

To build ctr, run 'make' inside the directory where you have the source.

```sh
$ make
```

## Examples
```sh
$ ctr -x 2f
Binary      : 101111
Octal       : 57
Decimal     : 47
Hexadecimal : 0x2f

$ ctr -b 1011001 -a
ASCII       : 'Y'
Binary      : 1011001
Octal       : 131
Decimal     : 89
Hexadecimal : 0x59
```
